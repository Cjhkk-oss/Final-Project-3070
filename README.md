# Disaster Prep App

A React Native + Expo mobile application for local disaster preparedness and response.

## Features
- Gamified emergency kit checklist
- Saved checklist progress using AsyncStorage
- Real-time weather information using Open-Meteo
- Nearby earthquake activity using USGS
- Searchable disaster guides and survival skills
- Nearest shelter lookup using GPS
- Emergency contact quick access

## Tech Stack
- React Native
- Expo
- Expo Location
- AsyncStorage

## Run locally
```bash
npm install
npx expo start
